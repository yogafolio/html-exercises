Mencoba Tag figure dan untuk menampilkan keterangan (caption) pada audio

Berikut ini adalah audio dengan Tag figure dan figcaption

diatas adalah hasil dari tag figure dan figcaption pada audio untuk menampilkan caption pada audio
Pada contoh ini caption berupa link menuju alamat sumber audio

Ket TAG : 	h1 = untuk membuat judul
		p = untuk paragrap
		figure = untuk mengelompokkan audio dengan hyperlink
		figcaption = untuk membuat keterangan audio (caption)
		audio = untuk menampilkan audio

ket ATRIBUT : 	src = untuk menentukan alamat file audio
		controls = untuk memunculkan menu control audio untuk pengaturan detail
		style = digunakan untuk mengatur width dan height audio (ukuran)
		a = untuk mendefinisikan sebuah link
