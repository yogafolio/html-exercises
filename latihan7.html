Mencoba Tag audio untuk menampilkan pemutar suara/musik

Berikut ini adalah musik dengan Tag audio

diatas adalah hasil dari tag audio

Ket TAG : 	h1 = untuk membuat judul
		p = untuk paragrap
		audio = untuk memunculkan audio/musik

ket ATRIBUT : 	src = untuk menentukan alamat file video
		controls = untuk memunculkan menu control video untuk pengaturan detail
		style = digunakan untuk mengatur width dan height video (ukuran)
