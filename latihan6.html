Mencoba Tag figure dan untuk menampilkan keterangan (caption) pada video/movie

Berikut ini adalah video/movie dengan Tag figure dan figcaption

diatas adalah hasil dari tag figure dan figcaption pada video untuk menampilkan caption pada video/film
Pada contoh ini caption berupa link menuju alamat sumber video

Ket TAG : 	h1 = untuk membuat judul
		p = untuk paragrap
		figure = untuk mengelompokkan video dengan hyperlink
		figcaption = untuk membuat keterangan video (caption)
		video = untuk menampilkan video/movie

ket ATRIBUT : 	src = untuk menentukan alamat file video
		controls = untuk memunculkan menu control video untuk pengaturan detail
		poster = berfungsi membuat tumbhnail untuk suatu video
		style = digunakan untuk mengatur width dan height video (ukuran)
		a = untuk mendefinisikan sebuah link
